# Portfolio Website Construction Plan: Sunita A-nita

## 1. Project Overview
Build a modern, minimalist portfolio website for a data analyst, hosted on GitHub Pages.

## 2. Design System
- **Theme:** Dark Mode (GitHub-inspired dark palette: `#0d1117` background, `#58a6ff` accent).
- **Typography:** JetBrains Mono (via Google Fonts) for a developer-centric aesthetic.
- **Layout:** Single-page responsive design using CSS Grid and Flexbox.

## 3. Project Structure
- `index.html` (Semantic structure)
- `style.css` (Dark theme, responsive layout, font implementation)
- `script.js` (Optional: minor interactivity)

## 4. Implementation Roadmap
1.  **Preparation:** Setup local directory `~/projects/portfolio`.
2.  **Layout Construction:** Develop semantic HTML skeleton.
3.  **Theming:** Define CSS variables, dark theme, and font setup.
4.  **Component Development:**
    - **Hero Section:** "Sunita A-nita | Data Analyst".
    - **Projects Section:** Grid layout for 3 data projects (Title, Description, Links).
    - **Contact Section:** Email/Links.
5.  **Refinement:** Final testing and responsiveness checks.
6.  **Deployment:** Configure Git, create public GitHub repository, push changes, and enable GitHub Pages in settings.

## 5. Verification
- Validate responsiveness across screen sizes.
- Verify fonts and dark theme colors.
- Ensure GitHub Pages live URL functionality.
